📝 Introduction

FinSight is a financial report analysis and forecasting system designed for SMEs, investors, and finance students. Users can upload financial reports (Balance Sheet, Income Statement, Cash Flow), and the system will automatically analyze key indicators, provide comparisons, and forecast future trends using AI/ML.

🎯 Objectives

Automate financial report analysis (liquidity, profitability, efficiency, leverage ratios).

Forecast revenue, profit, and cash flow with ML models (Logistic Regression, Decision Trees).

Enable cross-company and industry comparison.

Export PDF/Excel reports with interactive dashboards.

Support multiple languages (English/Vietnamese).

🚀 Key Features

📊 Financial Report Analysis: Import data from Excel/CSV or upload directly.

🔍 Cross-company Comparison: Evaluate financial ratios across companies.

📈 AI/ML Forecasting: Predict trends from historical data.

🗂 User Management: Roles for Admin, Analyst, Investor, Student.

🌐 User-friendly, multi-language interface.

📑 Report Exporting: PDF/Excel reports and visual dashboards.

👥 Target Users

SMEs: Gain insights into financial health without needing experts.

Investors & Analysts: Make faster, data-driven decisions and reduce risks.

Students & Lecturers: Learn and practice financial report analysis with real-world tools.

🛠 Technology Stack

Frontend: ReactJS, JavaScript

Backend: Python, C++

Database: MongoDB, MariaDB, Kafka, Hadoop

Others: Docker, MongoDB Atlas, Postman, Selenium, W&B, Figma, Trello

📅 Development Plan

Duration: Aug 19, 2025 – Dec 10, 2025 (12 weeks, 4 sprints)

Methodology: Agile/Scrum

⚠️ Risks & Mitigation

System slowdown with many users → optimize DB, caching, load testing.

Forecasting inaccuracies → update datasets, apply cross-validation.

Data security risks → encryption, two-factor authentication, security testing.

Complex UI → simple, intuitive design tested with SMEs and students.

📦 Expected Outcomes

A practical, easy-to-use financial analysis & forecasting platform.

Faster and more accurate investment decision-making.

Academic value for students and practical value for businesses.
